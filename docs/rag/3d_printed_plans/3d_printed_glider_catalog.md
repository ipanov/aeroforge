## CONSTRUCTION TECHNIQUE REFERENCE

### 1. Internal Structure Approaches

| Approach | Used By | Description | Pros | Cons |
|----------|---------|-------------|------|------|
| **Vase mode / spiralize** | Eclipson, Floonda, VasePlane, Tom Stanton | Single continuous wall, no infill | Lightest possible, no stringing/oozing | Weakest, single layer only |
| **Vase mode + internal ribs** | Tom Stanton, Bug Lite | Diagonal grid ribs printed as part of vase mode spiral | Light + structural, ribs as continuous line | Complex CAD setup (quadrant splitting) |
| **Gyroid infill (low %)** | Blanik L-13, Planeprint Aeron | 5-15% gyroid infill between walls | Good strength-to-weight, isotropic | Heavier than vase mode |
| **Shell + carbon spar** | Kraga Kodo, most designs | 0% infill shell with external carbon tube reinforcement | Simple, proven, adjustable stiffness | Requires separate carbon hardware |
| **D-box + carbon** | SoarKraft | Printed D-box forward section with pultruded carbon | Excellent torsional rigidity, realistic structure | More complex printing |
| **Standard walls + infill** | 3DLabPrint, most paid designs | Multiple walls with strategic infill | Strongest, most durable | Heaviest |
| **LW-PLA foaming** | Planeprint, Bug Lite | Active foaming of LW-PLA at ~230-250C | 50-70% weight reduction vs PLA | Requires specific filament and temperature tuning |

### 2. Hinge Mechanisms

| Type | Used By | Description |
|------|---------|-------------|
| **Print-in-place** | Eclipson Vortex | Aileron and hinge axis printed as single piece. Functional off the print bed. |
| **TPU living hinge** | Argon 1500, various | TPU strip glued or slotted into joint. Flexible but durable. |
| **Wire/paper-clip pin** | Painless360, Kraga | Straightened paper clip or wire as hinge pin through printed loops. Simple, cheap. |
| **CA (cyanoacrylate) hinges** | Nucking Futs, traditional | Standard RC CA hinges inserted into slots. Proven, strong. |
| **Covering film hinge** | Kraga Kodo, various | Covering film (Oracover etc.) acts as hinge across gap. Very light. |
| **Balsa sheet hinge** | Crobe | Thin balsa trailing edge naturally flexes. Simple but limited deflection. |

### 3. Spar Integration Methods

| Method | Used By | Description |
|--------|---------|-------------|
| **Through-hole for carbon tube** | Most designs | Printed sleeve/channel through wing for carbon tube slide-in |
| **D-box with pultruded carbon** | SoarKraft | Forward wing section forms D-box around carbon strip |
| **Glued-in carbon rod** | Binio3D, various | Carbon rod CA-glued into printed channel |
| **Wing joiner tube** | Jart 3DP, larger designs | Separate carbon tube spanning fuselage as wing joiner |
| **No spar (shell only)** | VasePlane, some vase mode | Airfoil shape alone provides rigidity. For non-RC or light loads only. |

### 4. Panel Joint Methods (Multi-Part Wings)

| Method | Used By | Description |
|--------|---------|-------------|
| **Male-female friction fit** | Eclipson (all models) | Printed male stub slides into female socket. Precise alignment. |
| **Carbon tube joiner** | Planeprint, Jart 3DP | Carbon tube passes through both panels. Strong, allows disassembly. |
| **Screw/bolt connection** | Kraga Kodo II | M2/M3 screws through alignment tabs. Positive lock. |
| **Magnet locks** | Floonda | Neodymium magnets for quick wing attachment/removal. |
| **Glued joint** | Binio3D | Permanent glue joint between panels. Strongest but non-removable. |

### 5. Vase Mode Printing Tricks

1. **Tom Stanton's quadrant method**: Model internal ribs as separate body, split into quadrants. When combined with outer shell, slicer treats entire print as continuous external perimeter = true vase mode with internal structure.

2. **Nose weight balancing**: Print nose cone separately with adjustable infill percentage to tune CG.

3. **Extrusion width override**: Manually increase extrusion width in slicer (e.g., 0.6-0.8mm with 0.4mm nozzle) for stronger layer adhesion in vase mode.

4. **Swept leading edge requirement**: Horizontal leading edges need supports. Sweep the LE to avoid support requirement in vase mode.

5. **Straight trailing edge requirement**: Vase mode spiral cannot reach swept-back trailing edges. Keep TE straight or manage the spiral path carefully.

6. **No retractions**: Vase mode prints without retractions = continuous strong layer bonding. Critical for LW-PLA which oozes badly with retractions.

7. **LW-PLA temperature zones**: Lower temp (~200C) for dense/strong sections, higher temp (~240C) for maximum foaming/weight reduction. Some slicers allow per-layer temp changes.

## KEY OBSERVATIONS FOR AEROFORGE

### Designs Most Relevant to Our 2560mm Sailplane

1. **Planeprint Rise** (2350mm, 650g, 19 g/dm2) - Closest to our target in size and philosophy. Their LW-PLA + carbon rod approach is proven. F5J-class performance.

2. **Planeprint Aeron** (2300mm, 1680g, gyroid infill) - Shows what a more robust 3D printed glider looks like. V-tail is interesting.

3. **Planeprint ASK14** (2500mm, 1700g) - Similar wingspan to ours. Uses PLA+LW-PLA+TPU mix. Retractable wheel is impressive.

4. **Eclipson Apex** (2300mm) - Hybrid PLA+/carbon/LW-PLA construction. Male-female fitting system is elegant.

5. **Blanik L-13** (2400mm, 1400g) - Uses gyroid infill at 6%. 9 servos. Close to our complexity level.

### Construction Lessons

- **Wing loading target**: Best thermal gliders achieve 19 g/dm2 (Planeprint Rise). Our target of ~29 g/dm2 is reasonable for a heavier airframe.
- **LW-PLA is mandatory** for competitive thermal performance at this size.
- **Vase mode with internal ribs** (Tom Stanton method) could be revolutionary for our panel approach.
- **Print-in-place hinges** (Eclipson Vortex) eliminate separate hinge hardware.
- **Gyroid infill at 5-15%** provides isotropic strength for non-vase-mode parts.
- **Carbon tube through-holes** are universal for spar integration.
- **Male-female panel joints** (Eclipson) + carbon tube joiners (Planeprint) are the two proven approaches.

### What Nobody Has Done (Our Opportunity)

- **Blended airfoil at every rib** - All designs use constant airfoil per panel at best
- **Geodesic/lattice internal structure** - Only basic gyroid infill explored
- **Parametric source CAD** - Only Jart 3DP offers STEP files. Nobody offers fully parametric models
- **AI-optimized twist distribution** - All designs use fixed washout angles
- **Topology-optimized mounts** - Nobody does this for servo/hardware mounts
- **10-panel continuous blend** - Most use 2-4 wing panels maximum
