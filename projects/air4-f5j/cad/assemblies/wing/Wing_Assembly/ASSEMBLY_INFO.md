# Wing_Assembly

## Purpose

Top-level full-wing parent assembly. This drawing exists so the wing can be reviewed and iterated
as a whole before work is pushed down into wing-half and panel components.

## Why This Was Added

The repo had strong wing research and panel-level work, but it skipped the explicit assembly-first
drawing artifact required by the workflow. This file and the accompanying drawing correct that order.
