# Fuselage_Assembly

## Purpose

Top-level fuselage parent assembly. This is now the hard parent artifact above the fuselage sections
and below the full aircraft integration.

## Why This Matters

The repo already had strong fuselage research and section schedules, but it drifted into section-first
implementation before committing a full fuselage three-view. That order is now corrected here.

## Current Baseline

- Total length: 1046 mm
- Wing LE station: 260 mm
- HStab pivot station: 911 mm
- Integrated fin body: X=650 to X=1046
- 5 printed sections with 4x 2 mm CF longerons
