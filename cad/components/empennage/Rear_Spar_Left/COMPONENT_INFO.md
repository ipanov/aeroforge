# Component: Rear_Spar_Left

## Type
Off-shelf (carbon fiber rod)

## Description
2mm pultruded carbon fiber rod serving as the rear spar for the left H-stab half.
Positioned at 65% chord to prevent trailing edge flutter and provide torsional
stiffness to the all-moving stabilizer.

## Specifications

| Parameter | Value |
|-----------|-------|
| Diameter | 2mm OD |
| Length | ~200mm |
| Material | Pultruded carbon fiber |
| Mass | ~0.5g |
| Position | 65% chord |

## Sourcing
- HobbyKing, AliExpress, or local hobby shop (Bulgaria)
- Standard 2mm CF rod

## Assembly
Part of `HStab_Assembly` — glued into rear spar channel in HStab_Left.
