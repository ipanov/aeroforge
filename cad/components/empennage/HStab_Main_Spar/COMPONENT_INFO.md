# Component: HStab_Main_Spar

## Description
The main bending spar for the horizontal stabilizer. A pultruded carbon fiber tube that threads continuously through both stab half-shells and the VStab fin at 25% chord. Acts as the primary structural element resisting aerodynamic bending loads on the H-Stab. The tube cross-section (OD 3 / ID 2 mm) provides a high stiffness-to-weight ratio with a hollow bore that keeps mass minimal.
