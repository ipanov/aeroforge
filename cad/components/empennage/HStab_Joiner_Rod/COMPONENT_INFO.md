# Component: HStab_Joiner_Rod

## Type
Off-shelf (carbon fiber rod)

## Description
3mm pultruded carbon fiber rod serving as the pivot axis for the all-moving
horizontal stabilizer. Runs through both halves at 25% MAC, retained by
brass tube bearings in the PETG tail mount socket.

## Specifications

| Parameter | Value |
|-----------|-------|
| Diameter | 3mm OD |
| Length | ~430mm (full H-stab span) |
| Material | Pultruded carbon fiber |
| Mass | ~1.4g |
| Function | Pivot axis + structural joiner |

## Sourcing
- HobbyKing, AliExpress, or local hobby shop (Bulgaria)
- Standard 3mm CF rod, widely available

## Assembly
Part of `HStab_Assembly` — slides through pivot channels in HStab_Left and HStab_Right,
retained by E-clips or CA'd collars at the bearing points.
