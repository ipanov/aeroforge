# Component: HStab_Tip_Cap

## Description
Replaceable tip cap for the horizontal stabilizer fixed half (stab). Covers the
outermost 4.5mm of span (y=209 to y=213.5) where the superellipse planform
transitions through the C1-continuous cubic tip closure to zero chord. Bonded
with CA glue at a 1mm overlap joint (y=209-210). Designed to break on impact
during handling, protecting the main stab body from damage. Identical part
used on both left and right sides.

## Specifications
| Parameter | Value |
|-----------|-------|
| Type | Custom printed (LW-PLA) |
| Span coverage | y=209 to y=213.5mm (4.5mm) |
| Root chord at joint | 34.6mm (at y=209) |
| Tip chord | ~1.2mm (at y=213.5) |
| Airfoil | HT-13/HT-12 blend (same as stab body) |
| Mass | 0.05g each |
| Quantity | 2 (one per side, identical) |
| Bond method | CA glue at y=209-210 overlap |
| Alignment | 1mm dia cylindrical socket pin |

## Print Settings
| Setting | Value |
|---------|-------|
| Material | LW-PLA |
| Temperature | 230°C |
| Mode | Vase mode 0.45mm |
| Orientation | Flat, joint face down |

## Assembly Notes
- Slide tip cap onto stab end, aligning socket pin
- Apply thin CA to the 1mm overlap zone (y=209-210)
- Hold 10 seconds for bond
- If broken: twist off remnants, sand flat, glue replacement
- Keep 2-3 spares in field box
- The elevator tip is NOT separate — it's integrated into the elevator print
