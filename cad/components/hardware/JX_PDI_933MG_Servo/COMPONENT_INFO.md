# Component: JX_PDI_933MG_Servo

## Description

JX PDI-933MG digital micro servo with metal gears. Off-the-shelf component
used in the AeroForge sailplane for control surface actuation. Features a compact
form factor with mounting ears for M2 screws and a 21-tooth output spline.

## Specifications

| Parameter | Value |
|-----------|-------|
| Type | Off-shelf component |
| Model | JX PDI-933MG |
| Mass | 13 g |
| Torque | 3.5 kg-cm @ 6V |
| Voltage | 4.8-6.0V |
| Speed | 0.11 sec/60 deg @ 6V |
| Dimensions (L x W x H) | 23.0 x 12.2 x 29.0 mm |
| Ear-to-ear | 32.0 mm |
| Output spline | 21T standard micro |
| Wire | ~150mm, 3-pin JR connector |
| Mounting holes | M2 x2, ear-to-ear 32.0mm |
| Color | Dark blue/black case |

## Usage in Sailplane

Flaps (x2) - needs higher torque for crow braking

## Assembly Notes

- Servo mounts into a printed pocket matching the body dimensions (23.0 x 12.2 mm).
- Mounting ears rest on the pocket ledge; secured with M2 x 6mm screws.
- Output shaft protrudes through the skin; control horn attaches to the 21T spline.
- Wire routes through internal channels to the receiver bay.
- Pocket depth = body height below ears (17.4 mm from bottom to ear centerline).
